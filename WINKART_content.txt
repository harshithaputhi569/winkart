WINKART - Software Requirements & Technology Recommendation
Project Type: Mobile Commerce Application with seller Panel
1. Project Overview
WINKART is a mobile-based shopping platform where users can browse products, search and filter items, receive product recommendations, view promotional banners/videos, add products to cart, complete payments, and download invoices. sellers manage categories, products, banners, product structures, imports/exports, and recommendations.

Seller screen designs:
Login screen components (screen 1): ref standard seller panel login screens.
1) WINKART logo and seller panel branding.
2) email / mobile number input field.
3) password input field with show/hide toggle.
4) forgot password link.
5) login button.
6) register as new seller link.

Dashboard screen components (screen 2): ref shopify seller dashboard / Amazon seller central dashboard.
1) top header bar with shop name, notification bell icon, and seller profile avatar.
2) summary cards row - total orders today, total revenue today, pending orders, total products listed.
3) recent orders list with order id, customer name, amount, and status badge.
4) top selling products card with product image, name, and units sold.
5) revenue chart (weekly/monthly toggle) showing sales trend graph.
6) bottom navigation bar. (dashboard, products, orders, banners, profile) ref seller panel nav.

Category management screen (screen 3): ref Shopify product categories admin screen.
1) screen header with title “Categories” and add new category button (+).
2) search bar to filter categories by name.
3) multi-level category tree list - level 1 categories with expand/collapse arrow, nested level 2 sub-categories below each.
4) each category row shows: category icon/thumbnail, category name, product count badge, edit icon, delete icon.
5) add/edit category bottom sheet popup - category name field, parent category dropdown, category image upload, save button.

Product management screen (screen 4): ref Amazon seller central manage inventory screen.
1) screen header with title “Products” and add new product button (+).
2) search bar to search products by name or SKU.
3) filter tabs - all, active, inactive, out of stock.
4) product list cards - product thumbnail, product name, category, price, stock count, active/inactive toggle, edit icon.
5) add/edit product screen - product name, description, category selection (level 1 and level 2 dropdown), price, discount price, stock quantity, product images upload (multiple), product attribute mapping fields, save and publish button.

Banner and video management screen (screen 5): ref Shopify marketing banner management screen.
1) screen header with title “Banners & Videos” and add new banner button (+).
2) toggle tabs - image banners and video banners.
3) banner card list - banner preview thumbnail, title, scheduled date range, active/expired status badge, edit and delete icons.
4) add/edit banner bottom sheet - upload image or video file, banner title, start date picker, end date picker (for scheduling), target link or product selection, save button.
5) banner preview option to see how it looks on the user app before publishing.

Upselling/Cross-selling configuration screen (screen 7): ref Amazon seller backend product linking screen.
1) screen header with title “Upsell & Cross-sell”.
2) toggle tabs - upsell products and cross-sell products.
3) source product search - search and select the base product for which upsell/cross-sell items are to be configured.
4) linked products list - shows currently linked upsell or cross-sell products with remove option.
5) add products button to search and add new linked products from the product catalog.
6) save configuration button.

Import/Export screen (screen 8): ref standard bulk import/export admin screens.
1) screen header with title “Import & Export”.
2) import section - upload excel file (.xlsx) button, download sample template link, column mapping preview table, import now button.
3) export section - export type dropdown (products, orders, invoices), date range picker, export format selection (excel/csv), export now button.
4) import history log - list of previous imports with file name, import date, total records, success count, failed count, and download error report option.

Profile/Shop settings screen (screen 9): ref Shopify store settings screen.
1) seller profile section - shop logo image, shop name, seller name, registered mobile number, registered email id.
2) shop details section - shop description, shop address, shop location on map (pin location).
3) business hours section - day-wise toggle and open/close time picker for each day.
4) account settings section - change password option, notification preferences toggle, logout button.
5) save changes button at the bottom.
